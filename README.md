# 🌐 75 Hard Code Challenge: JavaScript Projects

This repository showcases all **19 front-end and Node.js projects** built using **JavaScript, HTML, and CSS**. The challenge focuses on web development fundamentals, DOM manipulation, asynchronous operations, and modern browser APIs.

## Challenge Focus

* **DOM Mastery:** Manipulating the Document Object Model for interactive UIs.
* **API Integration:** Utilizing `fetch` to consume external APIs and handle JSON data.
* **Local Storage:** Implementing client-side data persistence.

## Repository Structure

Most projects are contained within a single directory, often consisting of an HTML page, a CSS file, and the core JavaScript logic:

```bash
75HC-JS/
├── DXX_ProjectName/
│   ├── index.html          // The markup/structure
│   └── script.js           // The core JavaScript logic
└── README.md               // Overview and project tracking (this file)
```
